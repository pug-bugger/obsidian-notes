#### Use Terminal to Stop Dock Moving 
You can use a Terminal command to make the Dock "sticky" on the main display, only allowing it to move if you rest the mouse at the bottom of another screen for a long time.
1. Open **Terminal**.
2. Paste the following command and hit Enter:  
    `defaults write com.apple.Dock autohide-delay -float 999 && killall Dock`
3. _To reverse this later, use:_ `defaults delete com.apple.Dock autohide-delay && killall Dock`.
